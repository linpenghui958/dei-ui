## 介绍

在学习Vue过程中和朋友一起做的一个UI库。希望对你有用。

## 开始使用

1. 安装
    ```js
    npm install dei-ui
    ```
2. 引入

    ```js
    import { Button, Icon, ButtonGroup } from 'dei-ui-test'
    import 'dei-ui-test/dist/index.css'
    
    ```
